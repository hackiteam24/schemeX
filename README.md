#  SchemeX AI

An AI-powered platform that helps citizens discover, check eligibility, apply for, and track government welfare schemes through a simple and multilingual interface.

---

## ✨ Features

-  AI Chat Assistant
-  Eligibility Checker
-  Browse 4800+ Government Schemes
-  Online Scheme Applications
- Document Upload & Verification
-  Citizen Dashboard
-  Profile Management
-  Admin Dashboard
-  Multilingual Support
-  Voice Search

---

## 🛠 Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Python
- Django
- Django REST Framework

**Database**
- MySQL
